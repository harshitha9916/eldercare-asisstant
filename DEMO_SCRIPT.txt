[0:00 — HOOK + PROBLEM + WHY AGENTS — 35 seconds]
(Text on screen: "53M unpaid caregivers. 1.3M medication errors/year.")

NARRATION:
"53 million Americans care for an aging relative.
One missed medication dose can mean a hospitalization.
The problem isn't lack of caring — it's lack of coordination.

A traditional app doesn't solve this. It still forces caregivers
to navigate menus, remember to log things, and make every decision manually.

Agents are different. They understand natural language,
route tasks intelligently, and act autonomously on safe requests —
while stopping to ask a human before anything critical happens.

I built ElderCare Assistant — a multi-agent AI care coordinator
built on Google ADK 2.0."

---

[0:35 — ARCHITECTURE — 45 seconds]
(Show architecture_diagram.png fullscreen. Point to each layer.)

NARRATION:
"Every message flows through four layers.

First: a Security Checkpoint — pure Python, no LLM.
It scrubs phone numbers, SSNs, detects prompt injections,
and fires a 911 alert for emergencies — in milliseconds.

Second: the Orchestrator Agent understands intent
and delegates to the right specialist.

Three sub-agents handle the work — Medication, Routine, and Wellness.
Each connects to an MCP server and can only access the tools it needs.
The medication agent literally cannot schedule a doctor visit.

And before any critical action is committed —
the workflow pauses for caregiver approval.
The AI cannot approve itself. Ever."

---

[1:20 — LIVE DEMO — 105 seconds]
(Switch to ADK playground at http://127.0.0.1:18081)

NARRATION:
"Let me show this live.

[Type: Please add a new medication: Vitamin D3, 2000 IU, once daily in the morning.]

The security checkpoint passes it through.
The orchestrator delegates to the Medication Agent,
which calls the MCP tool — and the workflow stops.
'Caregiver approval required. Yes or no?'

I type yes. The medication is committed.

[Type: Log that I slept 7 hours, low pain today, mood cheerful.]

This one routes to the Wellness Agent and completes instantly —
no approval needed. The system knows the difference.

[Type: My father is having chest pain!]

No LLM is called at all.
The system intercepts this in pure Python and responds immediately.
In a real emergency, milliseconds matter."

[3:05 — SECURITY FEATURE — 25 seconds]

NARRATION:

The number was stripped before the LLM ever saw the text.
Full audit log written to Cloud Logging. Every request. Every time."
